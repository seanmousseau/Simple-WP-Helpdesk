# Security Policy

## Reporting a Vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Use [GitHub Security Advisories](https://github.com/seanmousseau/Simple-WP-Helpdesk/security/advisories/new) to report vulnerabilities privately. This keeps the details confidential until a fix is released.

We will acknowledge your report within 48 hours and work with you to assess and address the issue before any public disclosure.

## Scope

Vulnerabilities in `simple-wp-helpdesk/simple-wp-helpdesk.php` are in scope. Please include:

- A description of the vulnerability and its potential impact
- Steps to reproduce
- WordPress and PHP versions affected
- Any suggested remediation, if you have one
